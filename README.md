# Agentic-Al-Course-2025

Projects and notes from The Complete Agentic AI Engineering Course 2025 by ED Donner on Udemy. Learning to build autonomous AI agents with Ollama and Llama 3.2.

## Setup
- Install Ollama: [Download](https://ollama.com/download)
- Pull Llama 3.2: `ollama pull llama3.2:3b`
- Install Python dependencies: `pip install -r requirements.txt` (add file later)
